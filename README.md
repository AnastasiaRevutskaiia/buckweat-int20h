# buckweat-int20h

## Overview

Website for viewing prices for buckwheat

## Installation

```
npm install
```

```
npm run build
```

### Run

```
npm start
```

Application runs on <http://localhost:5000>

To run in dev mode:

```
npm run dev
```

In development mode client accessible on <http://localhost:3000>
